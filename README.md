# Firebird Updater

A lightweight utility for updating Firebird databases using SQL migration scripts.

## Features

- Detect current database version
- Apply pending SQL migration scripts
- Create timestamped backups
- Extract packaged builds (.7z)
- Simple Tkinter GUI
- Execution logging

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python firebird_updater.py
```

## Dependencies

- fdb
- py7zr
